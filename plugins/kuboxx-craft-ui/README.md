# Kuboxx Craft UI (Cursor plugin)

This directory is a **Cursor Marketplace**–shaped plugin: one [Agent Skill](https://agentskills.io/) that teaches assistants how to use [`@kuboxx/craft-ui`](https://www.npmjs.com/package/@kuboxx/craft-ui).

The skill source of truth is `cursor-skills/craft-ui-library-usage/SKILL.md` at the repository root. This plugin’s `skills/craft-ui-library-usage/SKILL.md` is a symlink to that file.

## Local tryout

Point Cursor at this plugin folder using the workflow described in the [Cursor plugin docs](https://cursor.com/docs/plugins) (or validate against [cursor/plugin-template](https://github.com/cursor/plugin-template)).
